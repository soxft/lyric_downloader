一款基于php的歌词文件批量下载器
## 使用方法
1.下载该文件/注意您的电脑需要配置有php的运行环境<br />
2.在命令行中输入php index.php 目录 (如: php index.php ./music)<br />
3.等待下载
## 注意
请使用php-cli而不是在网页中打开!
